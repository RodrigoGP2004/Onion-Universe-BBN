# Onion-Universe-BBN
Numerical integration of Big Bang Nucleosynthesis for the Onion cosmological model. It also allows for computations on standard LCDM varying the baryon-to-photon ratio. To compile, execute:

```bash
gcc -O3 -march=native bbn_c.c -o bbn_c -lm
````

To run the program, use:

```bash
./bbn_c
```
